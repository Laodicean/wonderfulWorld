allChess is a cross-compatible, multiplayer chess program, which places emphasis on holding a low footprint. Writen in Python and PyQT, it is very stable and fast.

Downloads

Binary - Windows: allChess.exe - 7.3 MB

Source - Windows/Mac/Linux (ZIP): allChess.zip - 732 KB
Dependancies (required for source installation):
    Python 3.3
    PyQT 4. Be sure to get the version for python 3.3.

Installation

If you are using the binary executable, allChess requires no installation or setup. Simply run the application, and allow network access if your operating system asks.

If running from source, you will need to make sure that you have python 3.3 and PyQT 4 installed before running the allChess.py script to run the program. As with the binaries, you may need to give permission for network mutiplayer features to work properly

Usage and Help

To play a game of chess locally (On the one computer) all you need to do is open allChess. The default play mode is local. If you want to start a new game, you can do so by selecting the New Game option from the file menu.

If you want to play against a friend over a network, you will need to know the IP address of one of the computers. On that computer, in allChess, select Options > Connect, and then click Host. On the other computer, navigate once again to Options > Connect, however this time, enter the host's IP in the address field. Now click connect. If successful, you should recieve feedback in the Info + Chat text area that you have connected to a partner. You may now chat, and play chess! Once a game has finished, either player may start a new one with File > New game. You can also resign, offer a draw, or disconnect from your opponent through the Options menu. Remember, these features are only available in multiplayer.

If you wish, you may change the board orientation with Options > Change Orientation. You may also change your control scheme from 2 Click to Select and Move which changes the cursor to the piece you select when you play the game. This can be done under Options > Control Scheme.
